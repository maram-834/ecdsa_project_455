About ECDSA

The **Elliptic Curve Digital Signature Algorithm (ECDSA)** is a cryptographic algorithm used for digital signatures. It is based on elliptic curve mathematics and provides a secure way to authenticate and verify data.

About the Project

This project implements an ECDSA-based tool that allows users to:

  • Generate private and public keys
  • Sign messages using the private key
  • Verify the authenticity of signed messages using the public key
